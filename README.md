## Environment configuration

Download and run the installation [NodeJS](https://nodejs.org/en/).

## Installation

```sh
$ cd forex-console
$ npm install
```

## Start the Server
Start server
```sh
$ npm start
```

## Running Test

Run the URL: http://localhost:8080/

## Running Test code
```sh
$ npm run test
```

## Build source
```sh
$ npm run build
```
